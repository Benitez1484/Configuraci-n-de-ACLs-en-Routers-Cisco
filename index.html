Objetivo del Tutorial

¿Qué aprenderás?

En este tutorial completo aprenderás a implementar y configurar Listas de Control de Acceso (ACLs) en routers Cisco para controlar el flujo de tráfico en una red. Dominarás las técnicas para permitir o denegar el acceso de dispositivos específicos a recursos de red, fortaleciendo la seguridad y gestionando permisos de manera eficiente.

Escenarios de Configuración

Exploraremos tres escenarios prácticos donde configuraremos ACLs para controlar el acceso de diferentes PCs a un servidor central:

1
Acceso Exclusivo PC0

Configuraremos ACLs para permitir que únicamente la PC0 (192.168.1.10) pueda acceder al servidor, mientras que PC1 y PC2 serán bloqueadas.

2
Acceso Exclusivo PC1

Modificaremos la configuración para otorgar acceso exclusivo a PC1 (192.168.1.11), denegando el acceso a PC0 y PC2.

3
Acceso Exclusivo PC2

Finalmente, configuraremos las ACLs para que solo PC2 (192.168.1.12) tenga acceso al servidor, bloqueando a las demás PCs.
